# NER model checkpoints for anonymization of Icelandic patient-to-clinic messages

This is a set of checkpoints for a NER model trained on **synthetic** Icelandic messages from patients to clinics.
The data used for training the model can be found here: https://github.com/jmicota/thesis-datasets-ner-icebert-iob2
